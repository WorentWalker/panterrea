## 📖 Где взять ключи

### Stripe:
1. https://dashboard.stripe.com/
2. Developers → API keys
3. Копируйте Publishable key и Secret key

### AWS S3:
1. https://console.aws.amazon.com/
2. IAM → Users → Create Access Key
3. S3 → Buckets → Create bucket

## 🔧 Если не работает

### Ошибка "Undefined constant" исчезла, но функции не работают

**Причина:** Используются тестовые ключи-заглушки

**Решение:** Добавьте реальные ключи (см. выше)

### Ошибка "Invalid API Key"

**Причина:** Неправильный Stripe ключ

**Решение:** 
1. Проверьте скопировали ли правильный ключ
2. Убедитесь что нет пробелов до/после
3. Проверьте режим (test/live)

### Изображения не загружаются в S3

**Причина:** Неправильные AWS ключи или права

**Решение:**
1. Проверьте Access Key и Secret Key
2. Проверьте что IAM пользователь имеет права на S3
3. Проверьте название bucket
