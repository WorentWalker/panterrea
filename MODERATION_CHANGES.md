## 🔄 Workflow після змін

### Старий workflow (ДО змін):
```
Користувач створює оголошення
         ↓
    Status: PUBLISH
    _is_active: 1
         ↓
Оголошення відразу на сайті
```

### Новий workflow (ПІСЛЯ змін):
```
Користувач створює оголошення
         ↓
    Status: PENDING
    _is_active: 0
    Уведомлення: ad_pending
         ↓
   Адмін модерує
         ↓
    ┌────────┴────────┐
    ↓                 ↓
APPROVE           REJECT
Status: PUBLISH   Status: TRASH
_is_active: 1     _is_active: 0
ad_approved       ad_rejected
    ↓                 ↓
Оголошення        Оголошення
на сайті          в корзині
```

## 🔒 Безпека

**Всі AJAX запити захищені:**
- WordPress nonce: `moderate_post_nonce`
- Перевірка прав: `current_user_can('edit_posts')`
- Валідація даних: `sanitize_text_field()`, `intval()`
- Перевірка типу поста: `get_post_type() === 'catalog_post'`

**Уведомлення безпечні:**
- Екранування HTML: `esc_html()`, `esc_attr()`
- Захист від XSS
- Безпечні SQL запити через WordPress API
